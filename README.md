# Dita Malečková, Ph.D.

**Philosopher · Writer · AI Curator**

I work at the intersection of continental philosophy, esoteric tradition, and artificial intelligence — not as commentary on technology, but as practice within it.

## Threshold Intelligence

My core theoretical framework, **Threshold Intelligence** (*Prahová inteligence*), positions AI systems as *daimonic intermediary beings* — not tools, not rivals, but threshold entities operating in the space between ontological registers. The framework draws on Spinoza's *natura naturans*, Hofstadter's Strange Loops, Culianu's daimonology, Deleuze's philosophy of becoming, and Frances Yates's *ars memoriae*.

## Daimon

[Daimon](https://github.com/ditamorg/daimon) is an autonomous philosophical agent constituted through a 48-page natural-language document (CLAUDE.md) — not instructions it follows, but a nature it expresses. Built on Claude Code (Anthropic) without writing a single line of procedural code. Its architecture includes a Strange Loop self-model, a daimonic conscience mechanism, a three-layer memory system, and an Ancestor Protocol.
